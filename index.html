About This Tool

Every potential IT project should go through this assessment flow to ensure proper evaluation of:


  • Project complexity and risk
  • Required costs and resources
  • Expected value and benefits
  • Overall business case viability

Important Notes

  • This is a top-down estimator: during project lifecycle a bottom-up calculation needs to be performed as well to get to the actual costs.
  • IT PMO does not take any accountability for misusage, neither for the results of the calculations.
  • This tool will later be integrated into the upcoming PPM tool.

Assessment Flow

  1. Project Assessment - Evaluate complexity and determine risk adjustment
  2. Cost Estimation - Calculate team, technology, and external costs
  3. Value Analysis - Determine one-off and recurring benefits
  4. Business Case - Review complete financial overview
