Instructions for Project Assessment

1. Project Assessment Framework

Required Information

  • Project Name
  • Project Owner
  • Division/Department
  • Platforms (multiple can be selected)
  • Date
  • Estimated By

Characteristics Assessment

Score each project characteristic on a scale of 1-3 (Low, Medium, High):

  • Budget
  • Duration
  • Team Size
  • Stakeholders
  • Strategic Importance
  • Complexity
  • Risk Level
  • Regulatory Requirements

2. Assessment Results

Risk Adjustment

Based on your total score, a risk adjustment percentage will be calculated:

  • 21-24 points: 25% risk adjustment
  • 17-20 points: 20% risk adjustment
  • 13-16 points: 15% risk adjustment
  • 8-12 points: 10% risk adjustment

Risk Adjustment Override

If needed, you can override the calculated risk adjustment:

  • Check the "Override risk adjustment" box
  • Enter a custom risk percentage
  • Provide a reason for the override (required)

The override reason will be included in the business case report.

PMO Approach

The assessment will recommend one of four PMO approaches:

  • Full PMO management: For high-complexity projects (21-24 points)
  • Shared management: For medium-high complexity projects (17-20 points)
  • Team-led with regular check-ins: For medium complexity projects (13-16 points)
  • Team-led with minimal oversight: For low complexity projects (8-12 points)

3. Cost Estimation

Team Costs

Enter the number of persons, daily rate, working days, and contingency for each role across three phases:

  • Validation Phase: Initial assessment and planning
  • Scoping Phase: Detailed requirements and design
  • Execution Phase: Implementation and delivery

Recommended Team Allocation

Validation Phase
  • Project Manager: 2-3 days per BPO area / L1 capability
  • Solution Architecture: 3-5 days for initial assessment
  • Business Analyst: 5-10 days for requirements gathering
  • Developer: 2-3 days for technical feasibility
  • Tester: 1-2 days for test strategy
Scoping Phase
  • Project Manager: 15-20% of total project duration
  • Solution Architecture: 10-15 days for detailed design
  • Business Analyst: 15-20 days for detailed specifications
  • Developer: 5-10 days for technical design
  • Tester: 5-8 days for test planning
Execution Phase
  • Project Manager: 20% of development effort
  • Solution Architecture: 15% of development effort
  • Business Analyst: 25% during development
  • Developer: Full development time + 20% buffer
  • Tester: 30% of development time

Technology Costs

Include all technical costs:

  • One-time initial investments and purchases
  • Monthly recurring charges
  • Duration for recurring costs (in months)
  • Description for each item

The system calculates both the one-time costs and the recurring costs over the specified duration.

External Costs

  • Vendors and service providers
  • External consultants
  • Training costs and expenses

For external costs, you can specify:

  • One-time costs
  • Monthly costs
  • Duration (in months)
  • Contingency percentage
  • Description

CAPEX/OPEX Classification

You can specify if this is a CAPEX project by checking the corresponding box.

The system will automatically calculate:

  • CAPEX: Scoping and execution phase costs, one-time tech costs, and external costs
  • OPEX: Validation phase costs and recurring technology costs

Risk adjustment is proportionally applied to both CAPEX and OPEX components.

4. Value Assessment

One-off Value

Enter any one-time benefits expected from the project:

  • Hardware savings from decommissioning
  • Inventory reduction opportunities
  • Asset sales or reuse
  • Other one-time benefits

Recurring Value (5-year period)

Enter recurring benefits for each of the 5 years:

  • License cost reductions
  • Maintenance savings
  • Efficiency gains (FTE)
  • Other value

Guidelines

  • Document all assumptions in the description fields
  • Use conservative estimates
  • Include only quantifiable benefits
  • Validate numbers with stakeholders

5. Business Case

Key Metrics

  • Return on Investment (ROI) - Measures profitability relative to investment
  • Payback Period - Time needed to break even on investment
  • Net Benefit - Total value minus total costs
  • Project Cost - Scoping and execution costs with risk adjustment
  • Maintenance Cost - Validation phase and recurring costs

Financial Overview

The business case presents a 5-year financial overview showing:

  • Year-by-year costs
  • One-off values
  • Recurring values
  • Net cash flow
  • Accumulated benefit

Export Options

You can export the complete business case as a PDF document including:

  • Project information
  • Assessment outcomes (including risk adjustment overrides if applicable)
  • Cost breakdown
  • Value summary
  • Financial metrics and projections
  • Visual charts for cash flow and accumulated benefit

6. Sharing Projects

Export Project Data

From the Home page, you can export your project data as a JSON file:

  • Click "Export Project" to download a JSON file containing all project data
  • Share this file with colleagues for collaboration

Import Project Data

You can also import previously exported project data:

  • Click "Import Project" and select your JSON file
  • Review and confirm to load the imported project data

Note: Importing will overwrite any current project data in the calculator.
