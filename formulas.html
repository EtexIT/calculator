Project Assessment Formulas

Score Calculation

Total Assessment Score

Total Score = Sum of all characteristic scores

Where:

  • Each characteristic is scored 1, 2, or 3 points
  • Minimum possible score: 8 (all characteristics scored as 1)
  • Maximum possible score: 24 (all characteristics scored as 3)

Risk Adjustment Determination

Risk Adjustment Percentage

Risk Adjustment = {
  25% if total score ≥ 21
  20% if 17 ≤ total score < 21
  15% if 13 ≤ total score < 17
  10% if total score < 13
}

PMO Approach Determination

Management Approach Selection

PMO Approach = {
  "Full PMO management" if total score ≥ 21
  "Shared management" if 17 ≤ total score < 21
  "Team-led with regular check-ins" if 13 ≤ total score < 17
  "Team-led with minimal oversight" if total score < 13
}

Application in Cost Calculations

Adjusted Project Cost

Adjusted Cost = Base Cost × (1 + Risk Adjustment Percentage)

Example:

  • Base Cost: €100,000
  • Risk Adjustment: 20%
  • Adjusted Cost = €100,000 × (1 + 0.20) = €120,000

Team Cost Calculations

Per Role Cost Formula

Role Cost = Daily Rate × Number of Days × Number of Persons × (1 + Contingency%)

Applied separately for each phase (Validation, Scoping, Execution)

Technology Cost Calculations

One-time Technology Costs

One-time Cost = Sum of all one-time investments

Recurring Technology Costs

Recurring Cost = Monthly Cost × Number of Months

External Cost Calculations

External Cost with Contingency = Base Cost × (1 + Contingency%)

Risk Adjustment

Risk Adjusted Cost = Base Cost × (1 + Risk Adjustment%)
Where Risk Adjustment% is based on Assessment Framework score:
  • 21-24 points: 25% adjustment
  • 17-20 points: 20% adjustment
  • 13-16 points: 15% adjustment
  • 8-12 points: 10% adjustment

CAPEX/OPEX Calculations

CAPEX Components

CAPEX = Scoping Phase Costs + Execution Phase Costs + One-time Tech Costs + External Costs

OPEX Components

OPEX = Validation Phase Costs + Recurring Tech Costs

Risk Distribution

Risk Amount Distribution = Total Risk × (Component Cost / Total Cost)

Value Calculations

One-off Value Total

Total One-off Value = ∑(Hardware savings + Inventory reduction + Asset sale)

Yearly Recurring Value

Yearly Value = ∑(License reductions + Maintenance savings + Efficiency gains + Other value)
Calculated separately for each year (Years 1-5)

Total Recurring Value

Total Recurring Value = ∑(Year 1 + Year 2 + Year 3 + Year 4 + Year 5)

Total Project Value

Total Project Value = Total One-off Value + Total Recurring Value

Value Flow Chart Calculation

Year 1 Total = One-off Value + Year 1 Recurring Value Subsequent Years = Yearly Recurring Value

Accumulated Value Calculation

Accumulated Value(Year N) = One-off Value + ∑(Recurring Value Years 1 to N)
Where N is the year number (1-5)

FTE Value Calculation Guide

FTE Value = (Hours Saved per Year × Average Hourly Rate) / Annual Working Hours
Where:
  • Annual Working Hours = 1,720 (standard FTE hours)
  • Average Hourly Rate should be based on local market rates
  • Hours Saved should be validated with process owners

Value Calculations

One-off Value Total

Total One-off Value = ∑(Hardware savings + Inventory reduction + Asset sale + Other one-time benefits)

Yearly Recurring Value

Yearly Value = ∑(License reductions + Maintenance savings + Efficiency gains + Process improvements + Cost avoidance)
Calculated separately for each year (Years 1-5)

Total Recurring Value

Total Recurring Value = ∑(Year 1 + Year 2 + Year 3 + Year 4 + Year 5)

Total Project Value

Total Project Value = Total One-off Value + Total Recurring Value

Business Case Metrics

Return on Investment (ROI)

ROI = ((Total Benefits - Total Costs) / Total Costs) × 100%
Where:
  • Total Benefits = Total Project Value over 5 years
  • Total Costs = Risk-adjusted total cost

Payback Period

Payback Period = Time until Accumulated Benefits ≥ Total Costs
Where:
  • Accumulated Benefits = Cumulative sum of benefits per year
  • Benefits include both one-off and recurring values

Net Benefit

Net Benefit = Total Project Value - Total Risk-Adjusted Cost

Efficiency Gains Calculation

FTE Value Calculation

FTE Value = (Hours Saved per Year × Average Hourly Rate) / Annual Working Hours
Where:
  • Annual Working Hours = 1,720 (standard FTE hours)
  • Average Hourly Rate based on local market rates
  • Hours Saved validated with process owners

Accumulated Benefit

Year-over-Year Accumulation

Accumulated Value(Year N) = One-off Value + ∑(Recurring Value Years 1 to N)
Where N is the year number (1-5)
